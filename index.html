Welcome

Hello, my name is Zsolt Nagy and I work as a Software Test Engineer in Budapest Hungary. I have been working professionally in software development since 2014 and I am currently working for one of the biggest telecommunication giants. I graduated as an electrical engineer at Budapest University of Technology and Economics. As an engineer I like to solve problems and build great products. When I was a kid I always played with lego and was obsessed with making new things, but lately my new passion is making things even better.

I have never met a programmer who never had a bugs in his/her code. Even the best ones are not able to write bug-free code because we are humans and humans make mistakes.

But we can do a lot to recognize these mistakes as early as possible, correct and learn from them and try our best to prevent them in the future. As a Software Test Engineer my duty is to ensure that we develop the highest possible quality products which serve our customers' needs the best.

In the near future I would like to participate in an open source project and start my professional blog about software testing.
